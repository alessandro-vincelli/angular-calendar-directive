# angular-calendar-directive
Angular Directive for Calendar with Google Material Design

## Demo

See the demo at http://synerzip.github.io/angular-calendar-directive

# Build 
- npm install 
- bower install

# Run App - Development Mode
- grunt serve

# Run App - Production Mode
- grunt serve:prod

# Build App
- grunt build
